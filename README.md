
# DM Meet - Sistema de Reuniões Online

Projeto base para sistema de reuniões online inspirado no Google Meet.

## Tecnologias
- React
- Vite
- Supabase
- Cloudflare Pages
- LiveKit

## Deploy
1. Suba para o GitHub
2. Conecte no Cloudflare Pages
3. Configure as variáveis:

VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_LIVEKIT_URL=

## Instalação
npm install
npm run dev
